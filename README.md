# Frutiger Aero UX Webpage

A modern web page project inspired by the Frutiger Aero design aesthetic, characterized by glossy textures, vibrant colors, and nature-inspired elements.

## About

This project recreates the distinctive visual style of the Frutiger Aero era (mid-2000s to early 2010s), featuring:
- Glossy, reflective surfaces
- Bright, vibrant color palettes
- Nature and environmental themes
- Smooth gradients and transparency effects
- Clean, optimistic design language

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Basic knowledge of HTML, CSS, and JavaScript

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/brandonorozco12623-creator/frutigeraero.uxwebpage.git
   ```

2. Navigate to the project directory:
   ```bash
   cd frutigeraero.uxwebpage
   ```

3. Open `index.html` in your web browser or use a local development server.

## Usage

Simply open the HTML files in your browser to view the webpage. For development, consider using a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.
